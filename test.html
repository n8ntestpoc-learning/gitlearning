Git Dashboard

Install Git, verify setup, and manage global vs repository-specific user configuration

Installing Git

  • Download Git installer from the official website
  • Install as --standalone (default installer)
  • If Git Bash tool is installed → it installs Git automatically
On Windows, Git Bash provides a Unix-like terminal along with Git CLI.

Verify Installation

Open Git Bash and type:

Verification Command
git

If Git is installed correctly, you will see a list of Git commands and usage help.

Verify Git Version

Open Git Bash and type:

Verification Command
git --version

Create File

nano filename.extension

Stage File

git add filename.extension

Commit File

git commit -m "Your commit message"

Check Log

git log

Repository-specific user (overrides global)

git config user.name "Your Name"
git config user.email "you@example.com"

Global Git user

git config --global user.name "Your Name"
git config --global user.email "you@example.com"

Repo Specific Git user

git config  user.name "Your Name"
git config  user.email "you@example.com"

Check current Git user (effective)

git config –list | grep-user

Specific User Related

git config user.name
git config user.email

Values Origin

git config --show-origin user.name
git config --show-origin user.email
